# Research Knowledge Map Display Contract Skeleton

Prompt 91 adds backend display-contract placeholders for the future Research
Knowledge Map display surface. This package is metadata-only, read-only, and
unavailable-by-default.

It does not implement active UI, frontend components, desktop components,
active knowledge map behavior, database access, persistent writes, traversal,
query, search, ranking, retrieval, embeddings, vector stores, paper parsing,
strategy generation, backtesting, recommendations, broker controls, or
execution APIs.
