# Research Artifact Index Boundary

Prompt 82 adds Research Artifact Index system boundary hardening only.

This package defines forbidden behavior registry contracts, endpoint boundary
policies, module boundary policies, cross-module invariant helpers, and health
metadata for the Research Artifact Index planning/API/display stack.

It does not implement indexing, search, ranking, retrieval, embeddings, vector
stores, ingestion, storage, upload/download/preview, paper parsing, strategy
generation, backtesting, recommendations, broker controls, active UI, or
execution APIs.
