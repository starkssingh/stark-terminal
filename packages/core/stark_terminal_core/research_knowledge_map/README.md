# Research Knowledge Map

Research Knowledge Map is a planning and guardrail package only. It defines
descriptive placeholders for future knowledge items, relationships, evidence,
provenance, lifecycle state, readiness, and safety guardrails.

It does not implement an active knowledge map, database, persistent writes,
traversal, search, ranking, retrieval, embeddings, vector store, paper parsing,
strategy generation, backtesting, recommendations, broker controls, or
execution APIs.
