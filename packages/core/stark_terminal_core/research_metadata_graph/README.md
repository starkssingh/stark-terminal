# Research Metadata Graph

Prompt 84 adds Research Metadata Graph planning and guardrails only.

This package contains placeholder contracts for future metadata graph nodes,
edges, provenance, lifecycle, dependency references, guardrails, readiness,
and health metadata. It does not implement a graph database, persistent graph
writes, graph traversal, graph query, graph search, graph ranking, graph
retrieval, embeddings, vector store, ingestion, storage, upload/download/
preview, paper parsing, strategy generation, backtesting, recommendations,
broker controls, readiness-to-trade, or execution APIs.
