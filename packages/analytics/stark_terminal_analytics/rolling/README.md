# Rolling Window Analytics

Prompt 28 implements descriptive rolling window analytics v0.

Allowed calculations are right-aligned rolling count, rolling mean, rolling min, and rolling max over validated synthetic/local vectors with source references. Results are descriptive-only research artifacts and keep signal, recommendation, DecisionObject, and execution flags false.

This package does not implement trend labels, thresholds, volatility, drawdown, correlation, indicators, feature computation, backtests, regimes, recommendations, decisions, broker integration, or execution APIs.
