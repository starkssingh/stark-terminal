# Correlation Analytics

Prompt 31 implements descriptive Pearson correlation analytics only.

The convention is sample covariance and sample variance using the n-1
denominator. Paired vectors must be finite, equal length, source referenced, and
must contain at least two observations. Correlation is undefined when either
vector has zero variance.

Correlation results are descriptive/research-only. They are not thresholds,
regime labels, signals, recommendations, DecisionObject evidence, or execution
instructions.
