# Volatility Analytics v0

This package contains descriptive volatility analytics for Prompt 29.

- Sample standard deviation uses the n-1 denominator.
- Population standard deviation uses the n denominator.
- Annualized volatility is descriptive only and requires explicit `periods_per_year`.
- Inputs require source references and synthetic/local/test posture.
- Outputs are descriptive-only research artifacts.
- No thresholds, regime labels, signals, recommendations, DecisionObject generation, or execution APIs are implemented.
