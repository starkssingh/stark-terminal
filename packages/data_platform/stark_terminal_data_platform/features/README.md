# Stark Feature Registry Foundation

This package contains the custom Stark Feature Registry foundation.

Prompt 10 creates metadata and governance contracts only:

- Feature definitions.
- Feature sets.
- Feature entities, values, and snapshots.
- Feature quality reports.
- Feature lineage records.
- In-memory registry and health checks.

Prompt 10 does not compute real features, implement indicators, train ML models, ingest market data, implement Feast integration, connect to external services, or create execution APIs. The custom registry is metadata/governance only. Feast may be integrated later through an explicit future prompt.
