# Research Artifact Registry Display Reference Placeholders

Prompt 72 reference display placeholders are descriptive references only.

They may describe placeholder labels and placeholder reference identifiers for future display contracts. They must not fetch URLs, read local files, validate checksums, display trusted source content, or show parsed paper excerpts.

The reference display boundary forbids:

- external calls, scraping, provider SDKs, and credential use
- file preview, file upload, file download, local file read, and active artifact ingestion
- paper ingestion, paper parsing, PDF parsing, arXiv ingestion, and LLM paper analysis
- strategy generation, backtesting, recommendations, confidence scoring, DecisionObjects, readiness-to-trade, broker controls, and execution APIs

References remain placeholder-only until future prompt approval and audit.
