# Research Artifact Provenance Placeholders

These are Research Artifact Registry planning and guardrails documents.

ResearchArtifactProvenancePlaceholder is descriptive only. It records
provenance identifiers, artifact identifiers, source type, source label, source
data reference placeholders, audit notes, schema_version, and created_at.

Provenance placeholders make no claim that source data is trusted real market
data. They do not validate external sources, make external calls, persist
artifacts, ingest files, parse papers, generate strategies, run backtests,
produce recommendations, or expose execution controls. There is no active
artifact ingestion, no active artifact storage, no paper parsing, no strategy
generation, no backtesting, no recommendations, and no execution APIs.

Development remains on Mac mini M2 / macOS / Apple Silicon. The target desktop
product remains Windows-native Stark Terminal.
