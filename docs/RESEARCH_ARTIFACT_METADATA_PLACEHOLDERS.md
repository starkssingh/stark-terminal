# Research Artifact Metadata Placeholders

These are Research Artifact Registry planning and guardrails documents.

ResearchArtifactMetadataPlaceholder captures artifact_id, artifact_kind, title,
description, tags, source_reference, provenance_reference, lifecycle_status,
schema_version, and created_at. It is a planning contract only.

Metadata placeholders do not include file contents, parsed paper text, strategy
logic, backtest metrics, recommendations, confidence scores, active
DecisionObjects, readiness-to-trade, broker controls, approvals, overrides, or
execution state. There is no active artifact ingestion, no active artifact
storage, no persistent artifact storage, no paper parsing, no strategy
generation, no backtesting, no recommendations, and no execution APIs.

Development remains on Mac mini M2 / macOS / Apple Silicon. The target desktop
product remains Windows-native Stark Terminal.
