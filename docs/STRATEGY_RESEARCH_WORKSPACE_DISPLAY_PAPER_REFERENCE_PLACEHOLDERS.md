# Strategy Research Workspace Display Paper Reference Placeholders

Prompt 65 defines paper visual placeholders for display contract metadata only.

## Paper Visual Placeholder Schema

Paper visual placeholders cover arXiv reference, DOI reference, local document
reference, abstract, and method summary visual kinds. They are unavailable by
default and cannot be treated as ingested or parsed papers.

## Boundary

No paper ingestion display, no paper parsing display, no PDF parsing, no arXiv
ingestion, no method extraction display, no strategy extraction display, no
code generation from paper, no backtest generation from paper, no
recommendation generation, no confidence scoring, no DecisionObject
generation, no readiness-to-trade, no broker controls, and no execution APIs
are implemented.

Development remains Mac mini M2 / macOS / Apple Silicon. The target desktop
product remains Windows-native Stark Terminal.
