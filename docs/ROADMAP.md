# Roadmap

- Milestone A - Foundation + institutional stack lock
- Milestone B - Core domain schemas + typed config
- Milestone C - PostgreSQL + TimescaleDB foundation
- Milestone D - DuckDB + Parquet research lake foundation
- Milestone E - Redis + Redis Streams + worker pipeline foundation
- Milestone F - ClickHouse + Feature Registry foundation
- Milestone G - Instrument master + market data contracts
- Milestone H - Market data ingestion + data quality
- Milestone I - Numerical/time-series analytics foundation
- Milestone J - Regime/state engine + DecisionObject generation
- Milestone K - Retail Decision Console + Quant Detail View
- Milestone L - Backtesting + risk analytics
- Milestone M - Options/F&O analytics
- Milestone N - ML research layer
- Milestone O - Paper-to-strategy lab
- Milestone P - Deployment hardening, Windows packaging, audits

Each milestone must preserve the safety boundary: research and decision support before automation, no broker execution, and no live order placement.
