# Research Artifact Index Display No Embeddings Vector Store Policy

Prompt 79 forbids embeddings and vector-store behavior in Research Artifact Index Display.

Display placeholders cannot expose embeddings, vector IDs, vector-store references, vector search, semantic search, embedding pipelines, or vector database dependencies.

Policy:

- No embeddings.
- No vector store.
- No embedding pipeline.
- No vector IDs.
- No vector-store references.
- No search engine.
- No ranking engine.
- No retrieval engine.
- No execution APIs.
