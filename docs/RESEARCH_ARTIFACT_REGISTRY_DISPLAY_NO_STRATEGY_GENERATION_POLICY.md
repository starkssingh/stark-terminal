# Research Artifact Registry Display No Strategy Generation Policy

Prompt 72 display contracts cannot generate or display generated strategies.

Forbidden behavior includes strategy generation, strategy code generation, signal generation, factor generation, alpha generation, generated thresholds, method-to-strategy conversion, paper-to-strategy conversion, and executable strategy code.

Display card placeholders and lifecycle badges must not imply a strategy exists, has been validated, or is ready to trade.

The display contract remains backend-only, read-only, unavailable-by-default metadata.
