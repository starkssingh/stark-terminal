# Strategy Research Workspace Display Dataset Reference Placeholders

Prompt 65 defines dataset visual placeholders for display contract metadata
only.

## Dataset Visual Placeholder Schema

Dataset visual placeholders cover synthetic dataset, local file, research
lake, and metadata visual kinds. They are unavailable by default and do not
display live or validated market data.

## Boundary

Dataset placeholders do not claim real market data, live data, validated
research datasets, validated backtest datasets, or execution datasets.
Synthetic/local references remain untrusted placeholders. Prompt 65 adds no
active UI, no frontend components, no desktop components, no paper ingestion,
no paper parsing, no strategy generation, no strategy code generation, no
backtesting, no optimization, no recommendation generation, no confidence
scoring, no DecisionObject generation, no readiness-to-trade, no broker
controls, and no execution APIs.

Development remains Mac mini M2 / macOS / Apple Silicon. The target desktop
product remains Windows-native Stark Terminal.
