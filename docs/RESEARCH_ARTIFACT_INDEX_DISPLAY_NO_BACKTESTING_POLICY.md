# Research Artifact Index Display No Backtesting Policy

Prompt 79 forbids backtesting in Research Artifact Index Display.

Display placeholders cannot display backtest metrics, backtest results, performance claims, optimization output, parameter search output, walk-forward analysis, validated strategy state, or backtested profitable state.

Policy:

- No backtesting.
- No optimization.
- No parameter search.
- No walk-forward analysis.
- No performance claims.
- No backtest result display.
- No recommendation.
- No readiness-to-trade.
- No execution APIs.
