# Research Artifact Registry Display No Recommendation Policy

Prompt 72 display contracts cannot generate or display recommendations.

Forbidden behavior includes research-as-recommendation, buy/sell/hold/watch/avoid active outputs, action generation, confidence scoring, active DecisionObject generation, active DecisionObject display, readiness-to-trade, and hidden trade interpretation.

Display card placeholders, provenance placeholders, lifecycle placeholders, and badges must not be interpreted as recommendation cards or decision-support outputs.

Recommendation generation remains forbidden.
