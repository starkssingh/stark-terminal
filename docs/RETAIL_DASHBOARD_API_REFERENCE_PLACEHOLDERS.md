# Retail Dashboard API Reference Placeholders

Retail Dashboard API reference placeholders keep Prompt 50 contract-only and
unavailable by default.

## Data Reference

The data reference placeholder names a future data source reference. It has no
real market data, no live data, no display-ready state, and no production data
claim.

## Decision Reference

The decision reference placeholder names a future decision reference. It has no
active DecisionObject, no recommendation availability, no action availability,
no confidence availability, no readiness-to-trade availability, and no
display-ready state.

## Safety Reference

The safety reference placeholder names a future boundary policy reference. It
has no safety pass, no approval granted, no override granted, no execution
allowed, and no broker controls allowed.

These placeholders do not create active UI, recommendation cards, action
generation, confidence scoring, DecisionObject generation, readiness-to-trade,
broker controls, approval, override, or execution APIs.

Development remains Mac mini M2 on macOS / Apple Silicon. The target desktop
product remains Windows-native Stark Terminal.
