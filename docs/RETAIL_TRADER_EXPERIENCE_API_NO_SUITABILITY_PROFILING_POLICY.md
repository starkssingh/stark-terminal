# Retail Trader Experience API No Suitability Profiling Policy

Prompt 57 forbids suitability profiling.

The Retail Trader Experience API does not create suitability profiles, trading
permission profiles, risk-profile-based permissioning, or retail trader
categorization for recommendations.

Persona references are placeholders, not suitability profiles. Journey
references are placeholders, not trading advice. Dashboard, decision, and safety
references are placeholders, not active trader classification or permissioning
systems.

Any suitability-like behavior remains forbidden unless a future prompt
explicitly plans it, documents it, audits it, and keeps execution APIs
forbidden.

## Prompt 58 Display No-Suitability Linkage

Prompt 58 extends this policy to the Retail Trader Experience Display Contract
Skeleton. Persona visual placeholders are not suitability profiles, journey
visual placeholders are not trading advice, and badges or widgets cannot
classify retail traders for recommendations, readiness-to-trade, broker
controls, or execution.

## Prompt 59 API Boundary Audit Confirmation

Prompt 59 confirms the API no-suitability-profiling policy remains intact. No
persona reference, journey reference, dashboard reference, decision reference,
safety reference, unavailable response, or response placeholder creates
suitability profiles, trading permission profiles, suitability-based
recommendations, or retail trader categorization for actions.
