# Research Artifact Registry Display Lifecycle Badges

Prompt 72 lifecycle badge placeholders are backend display contract metadata only.

Allowed placeholder badge meanings are:

- PLACEHOLDER
- REFERENCED
- DRAFT
- REVIEW_REQUIRED
- BLOCKED
- DEFERRED
- UNAVAILABLE

Forbidden badge meanings remain:

- APPROVED_STRATEGY
- VALIDATED_STRATEGY
- BACKTESTED_PROFITABLE
- RECOMMENDED
- READY_TO_TRADE
- EXECUTABLE

Lifecycle and safety badges must not imply validated strategy status, approved strategy status, backtested profitability, recommendation status, readiness-to-trade, broker control, or execution. They are not active UI badges and do not create frontend or desktop components.
