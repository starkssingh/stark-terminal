# Strategy Research Workspace Display Experiment Placeholders

Prompt 65 defines experiment visual placeholders for display contract metadata
only.

## Experiment Visual Placeholder Schema

Experiment visual placeholders cover experiment plan, validation plan, backtest
plan, walk-forward plan, and safety review plan visual kinds. They are
unavailable by default and non-executable.

## Boundary

No executable experiment, no executable backtest, no optimization, no
parameter search, no walk-forward analysis, no strategy execution, no
live-ready state, no recommendation-ready state, and no execution-ready state
exist. Prompt 65 adds no active UI, no frontend components, no desktop
components, no paper ingestion, no paper parsing, no strategy generation, no
strategy code generation, no backtesting, no recommendation generation, no
confidence scoring, no DecisionObject generation, no readiness-to-trade, no
broker controls, and no execution APIs.

Development remains Mac mini M2 / macOS / Apple Silicon. The target desktop
product remains Windows-native Stark Terminal.
