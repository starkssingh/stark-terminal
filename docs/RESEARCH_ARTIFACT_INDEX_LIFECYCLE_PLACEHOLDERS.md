# Research Artifact Index Lifecycle Placeholders

`ResearchArtifactIndexLifecyclePlaceholder` is a planning-only lifecycle/status contract. It may describe safe lifecycle states for future index planning.

## Allowed Statuses

- PLACEHOLDER
- REFERENCED
- DRAFT
- REVIEW_REQUIRED
- BLOCKED
- DEFERRED
- UNKNOWN

## Forbidden Status Meaning

No status may imply indexed, searchable, ranked, embedded, retrieved, validated strategy, approved strategy, backtested strategy, recommendation, readiness-to-trade, broker control, approval, override, or execution.

Lifecycle placeholders do not create an index engine, search engine, ranking engine, retrieval engine, embedding pipeline, vector store, persistent storage, ingestion, paper parsing, strategy generation, backtesting, recommendations, or execution APIs.
