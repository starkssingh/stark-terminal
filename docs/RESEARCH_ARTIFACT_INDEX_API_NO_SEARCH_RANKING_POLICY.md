# Research Artifact Index API No Search Ranking Policy

The Research Artifact Index API contract skeleton does not provide search,
ranking, retrieval, semantic search, keyword search, scoring, or result
ordering.

Request placeholders cannot execute search queries. Response placeholders
cannot return search results, ranking results, retrieval results, ranking
scores, or confidence scores.

No indexing engine, search engine, ranking engine, retrieval engine,
embeddings, vector store, ingestion/storage, file upload/download/preview,
paper parsing, strategy generation, backtesting, recommendations, broker
controls, readiness-to-trade, or execution APIs are introduced.
