# Research Artifact Index Display No Active UI Policy

Prompt 79 forbids active UI for Research Artifact Index Display.

Display contracts are backend-only placeholders. They do not create frontend pages, desktop UI, active widgets, active layouts, rendered cards, active filters, file previews, charts, or visualization dependencies.

Policy:

- No active UI.
- No frontend implementation.
- No desktop implementation.
- No active cards, widgets, layouts, pages, or artifact browser UI.
- No file preview.
- No broker controls.
- No execution APIs.

Future UI work requires a future prompt and audit. Windows-native Stark Terminal remains the target desktop product, but Prompt 79 does not implement desktop UI.
