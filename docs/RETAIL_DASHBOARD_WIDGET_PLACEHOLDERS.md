# Retail Dashboard Widget Placeholders

Retail Dashboard Display widget placeholders describe future widget categories without creating active widgets. They are display contract skeleton artifacts only and remain unavailable by default.

Widget kinds:

- generic placeholder
- data quality widget placeholder
- market context widget placeholder
- decision placeholder
- risk placeholder
- safety placeholder
- review placeholder
- unavailable

No widget placeholder is a recommendation widget. No widget placeholder is an action widget. No widget placeholder is a confidence widget. No widget placeholder displays an active DecisionObject. No widget placeholder generates DecisionObject output, readiness-to-trade, broker controls, approval controls, override controls, or execution APIs.

The placeholders are not active UI and do not create a frontend component or desktop UI component.

Development environment: Mac mini M2 / macOS / Apple Silicon.
Target desktop product: Windows-native Stark Terminal.
