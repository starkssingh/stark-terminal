# Research Artifact Index API No Backtesting Policy

The Research Artifact Index API contract skeleton contains no backtesting
engine, no optimization, no parameter search, no walk-forward analysis, no
performance claims, and no backtest result endpoints.

Request placeholders contain no backtest parameters. Response placeholders
contain no backtest results or performance metrics.

No indexing engine, search engine, ranking engine, retrieval engine,
embeddings, vector store, ingestion/storage, file upload/download/preview,
paper parsing, strategy generation, recommendations, confidence scoring,
readiness-to-trade, broker controls, or execution APIs are introduced.
