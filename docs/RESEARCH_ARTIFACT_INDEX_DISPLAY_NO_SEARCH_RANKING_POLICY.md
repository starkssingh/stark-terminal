# Research Artifact Index Display No Search Ranking Policy

Prompt 79 forbids search and ranking behavior in Research Artifact Index Display.

Display placeholders may show tag labels and reference labels only. They cannot run search, keyword search, semantic search, ranking, scoring, retrieval, ranking weight display, or active filter UI.

Policy:

- No search engine.
- No ranking engine.
- No retrieval engine.
- No semantic search.
- No keyword search.
- No search results.
- No ranking results.
- No retrieval results.
- No recommendations.
- No action generation.
- No confidence scoring.
- No execution APIs.
