# Research Artifact Index Key Placeholders

`ResearchArtifactIndexKeyPlaceholder` is a planning-only key contract. It may describe an artifact ID, registry ID, reference ID, provenance ID, tag, category, source label, or lifecycle status placeholder.

## Safety Contract

- `key_id` cannot be empty.
- UNKNOWN key kind is rejected.
- Keys are placeholders only.
- No lookup, search, ranking, storage, retrieval, fetch, local file read, external call, indexing engine, search engine, ranking engine, embedding pipeline, vector store, ingestion, paper parsing, strategy generation, backtesting, recommendations, broker controls, or execution.

Key placeholders are not index entries and are not persistent identifiers for live systems.
