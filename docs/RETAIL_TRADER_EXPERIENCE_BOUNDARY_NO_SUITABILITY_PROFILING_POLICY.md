# Retail Trader Experience Boundary No-Suitability-Profiling Policy

Prompt 61 confirms that suitability profiling remains forbidden across Retail
Trader Experience modules and endpoints.

The boundary layer forbids:

- Suitability profiling.
- Trading permission profiles.
- Persona-to-suitability-profile paths.
- Journey-to-trading-advice paths.
- Suitability-based recommendation behavior.
- Retail trader categorization for actions.
- Endpoint or module boundary bypasses that imply suitability permissioning.

Persona placeholders are not suitability profiles. Journey placeholders are
not trading advice. Display placeholders are not suitability displays. No
module or endpoint can bypass this boundary without a future prompt and audit.
