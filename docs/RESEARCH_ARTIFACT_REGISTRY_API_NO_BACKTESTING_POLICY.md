# Research Artifact Registry API No Backtesting Policy

Prompt 71 does not implement API backtesting, optimization, parameter search,
walk-forward analysis, performance claims, strategy validation, backtest result
storage, backtest result endpoints, or live/paper trading controls.

Backtest reference artifacts remain placeholders only and contain no metrics,
validation results, performance claims, generated strategies, recommendations,
confidence scores, active DecisionObjects, readiness-to-trade, broker controls,
or execution controls.

The API contract skeleton remains read-only and unavailable-by-default.

Development remains on Mac mini M2 / macOS / Apple Silicon. The target desktop
product remains Windows-native Stark Terminal.
