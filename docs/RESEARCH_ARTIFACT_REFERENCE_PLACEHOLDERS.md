# Research Artifact Reference Placeholders

These are Research Artifact Registry planning and guardrails documents.

ResearchArtifactReferencePlaceholder captures reference identifiers, artifact
identifiers, reference kind, URI placeholders, source labels, checksum
placeholders, schema_version, and created_at. Reference URI values are
placeholder-only.

The registry does not fetch URIs, validate external URLs, download files, read
files, parse papers, parse PDFs, ingest arXiv records, or run LLM paper
analysis. There is no active artifact ingestion, no active artifact storage, no
persistent artifact storage, no paper parsing, no strategy generation, no
backtesting, no recommendations, no broker controls, and no execution APIs.

Development remains on Mac mini M2 / macOS / Apple Silicon. The target desktop
product remains Windows-native Stark Terminal.
