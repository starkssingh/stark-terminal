# Strategy Research Dataset Reference Placeholders

Dataset reference placeholders define future research dataset metadata only.

## Planned Dataset Reference Kinds

- Synthetic dataset reference.
- Local file reference.
- Research lake reference.
- Metadata reference.

## Boundary

Dataset references do not claim real market data, live data, validated research datasets, validated backtest datasets, or execution datasets. Synthetic/local references remain untrusted placeholders. Prompt 63 adds no active UI, no paper parsing, no strategy generation, no backtesting, no recommendations, no broker controls, and no execution APIs.
