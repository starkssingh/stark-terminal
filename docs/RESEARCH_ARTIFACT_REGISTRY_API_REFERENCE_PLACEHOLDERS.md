# Research Artifact Registry API Reference Placeholders

Prompt 71 API references are descriptive placeholders only. They may identify
future artifact metadata, API reference, or provenance reference fields, but
they do not fetch external references, read local files, download files,
validate checksums beyond placeholder text, or claim that a source is trusted
real market data.

Reference placeholders are not artifact ingestion, not persistent storage, not
file upload/download behavior, not paper parsing, not PDF parsing, not arXiv
ingestion, not LLM paper analysis, not strategy generation, not backtesting,
not recommendations, not active DecisionObjects, not readiness-to-trade, not
broker controls, and not execution APIs.

The API remains read-only and unavailable-by-default until future prompts add
display contracts and safety audits.

Development remains on Mac mini M2 / macOS / Apple Silicon. The target desktop
product remains Windows-native Stark Terminal.
