# Research Artifact Index No Indexing Engine Policy

Prompt 77 does not introduce an indexing engine. Index metadata, keys, references, tags, provenance, and lifecycle records are placeholders only.

## Prohibited

- No build_index function.
- No run_indexing function.
- No persistent index writes.
- No artifact registry database tables.
- No artifact index database tables.
- No migrations.
- No object storage.
- No repository writes.
- No background indexing jobs.
- No artifact source fetching.

The Research Artifact Index is planning and guardrails only. Prompt 78 may add API contract skeletons only.
